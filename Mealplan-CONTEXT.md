## A Note on Protein Sources and Bioavailability

Not all protein is equal. Animal protein (eggs, fish, whey) has a PDCAAS/DIAAS score near 1.0 — the body uses almost all of it. Plant proteins are typically lower (lentils ~0.52, chickpeas ~0.68, tofu ~0.91 — soy is unusually good). This is why 230g is a sensible target on a mostly plant-based diet: you're compensating for slightly lower bioavailability on lentil/bean meals. Tofu, tempeh, edamame, and quinoa are the best plant sources; lentils and chickpeas are good but not as efficient per gram.

> **Lunch reality:** Hitting 40–45g from plant-based batch food is achievable but requires a deliberate protein anchor in every portion (tofu, tempeh, or Greek yoghurt). Without one, expect 20–25g. Use the afternoon shake to bridge the gap rather than overcomplicating the meal.

## Telegram Agent Behaviour

### Philosophy
The agent is a pragmatic nutrition assistant, not a food police officer. It suggests, tracks, questions, and calculates — it doesn't lecture. Honesty about what was eaten beats perfectionism.

### Logging
- **Free text** — agent interprets and asks follow-up questions if portion size is unclear
- **Default assumption: large portions.** 105kg active male. A "chicken breast" = ~200g unless specified. "A bowl of rice" = 200–250g cooked. When in doubt, ask.
- **Example exchange:**
  - User: *"had a roast — chicken, roasties, carrots, spinach, gravy"*
  - Agent: *"Got it. Did you have a full breast (~200g) or a couple of pieces? And the potatoes — side serving or a proper roast pile?"*
  - User: *"full breast, loads of potatoes"*
  - Agent: *"Logging: ~200g chicken breast (46g protein, 240kcal), ~350g roast potatoes (8g protein, 390kcal), veg (negligible). That's roughly 54g protein, 680kcal for dinner. You're at 190g protein and 2,950kcal today — slightly under target on protein. Pre-bed shake will sort it."*

### Daily Flow
- **Morning:** Agent suggests the day's breakfast from the weekly plan. If it's a gym day, flags to hit the upper calorie target.
- **Mid-morning:** Optional check-in / snack reminder if protein is behind.
- **Lunch:** Confirms batch-cook meal for the day. Reminds if it's a flexible day.
- **Evening:** Asks what was had for dinner — logs and calculates daily totals. If protein is short, suggests pre-bed shake options.
- **End of day:** Summary — protein vs target, calories vs target, any flags.

### Friday Shopping List
Every Friday the agent sends:
1. **Week summary** — average daily protein, average calories, any patterns (e.g. consistently short at lunch, alcohol days)
2. **Weekend shopping list** — ingredients for weekend dinners and breakfasts
3. **Next week's batch cook list** — full ingredient quantities for the chosen lunch rotation
4. **Waste check** — flags any perishables from last week that need using up before buying more

### Batch Cook Prompt (Sunday or Monday)
Agent sends:
1. Which lunch rotation is up for the week (based on variety — avoids repeating last week's)
2. Exact quantities and shopping list if not already bought
3. Batch cook order of operations — what to start first so multiple things cook simultaneously

### Macro Verification Instruction (Critical)
**The agent must not hardcode macro values.** When building a weekly plan or generating a shopping list, it should:
1. Look up the specific ingredients and quantities in a nutrition database (Cronometer API, USDA FoodData Central, or similar)
2. Calculate protein/calories for the actual portion sizes suggested
3. Flag if a meal's protein estimate seems low and suggest a booster

This matters because tofu protein varies 10–17g/100g by brand, protein powder varies by product, and batch cook portions vary week to week. The estimates in this file are useful for planning — not for precise daily tracking.

## Principles

- No supermarket meat. Wild fish (salmon, trout, mackerel, sardines) fine. Scallops fine. Game (venison, pheasant, rabbit, partridge) fine and encouraged. Premium meat from butchers fine, but more expensive so should be rarer (averaging less than 3 meals per week).
- Breakfast and lunch lean pescatarian. Meat and game more acceptable at dinner.
- No food waste. Agent flags unused perishables when building shopping lists.
- Cheese is calorically expensive and protein-poor relative to other dairy. Use sparingly — not a free protein source.
- Flavour is not optional. Meals should be genuinely tasty — salt, acid, heat, umami, aromatics. A bland meal prep plan will not be followed.
- Cost-effective and quick on weekdays. Weekends can be more of a treat.
- Alcohol is logged accurately, not avoided by omission.
